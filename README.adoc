== Repository Structure

This repository serves as the main hub for the OSSNetwork application. It is organized to bring together multiple components:

=== Components

* *frontend/* - Vue.js 3 + Vite frontend application (linked as git submodule)
* *backend/* - Python 3.12 + Django-Ninja backend API (linked as git submodule)
* *deployment/* - Contains deployment configurations, scripts, and documentation

=== Architecture

The application is split into separate repositories for better modularity and development workflow:

* Frontend repository will handle the user interface and client-side logic
* Backend repository will handle the server-side API and business logic
* This main repository acts as the integration point and contains deployment configurations

=== Getting Started

==== Clone the Repository with Submodules

To clone this repository along with all its submodules (frontend and backend):

[source,bash]
----
git clone --recurse-submodules https://github.com/dev-clement/OSSNetwork.git
----

==== Initialize Submodules in an Existing Clone

If you've already cloned the repository without submodules, you can initialize and fetch them:

[source,bash]
----
git submodule update --init --recursive
----

==== Update Submodules

To update the submodules to their latest commits:

[source,bash]
----
git submodule update --remote
----
